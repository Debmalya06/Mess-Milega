# Mess PG Sathi

A full-stack web application to help users find and manage Mess and PG (Paying Guest) accommodations.

## Tech Stack

**Frontend:**
- React + Vite
- Tailwind CSS
- React Router

**Backend:**
- Java Spring Boot
- Spring Security (JWT Authentication)
- MySQL

## Project Structure

```
├── client/          # React frontend
└── server/          # Spring Boot backend
    └── Mess_PgSathi/
```

## Getting Started

### Backend
```bash
cd server/Mess_PgSathi
./mvnw spring-boot:run
```

### Frontend
```bash
cd client
npm install
npm run dev
```

## Features

- User Authentication (Login/Register)
- Property Listings (Mess & PG)
- Search & Filters
- Booking Management
- Owner Dashboard
- Reviews & Ratings
- Virtual Tours
- Real-time Chat

## API Documentation

API collection available at `server/Mess_PgSathi/postman/`
